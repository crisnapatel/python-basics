# Loops

## For Loop
```python
# Iterate over a list
compounds = ["methane", "ethane", "propane"]
for compound in compounds:
    print(compound)

# Range for numerical loops
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4
```

## While Loop
```python
concentration = 1.0
while concentration > 0.1:
    concentration = concentration * 0.5
    print(f"C = {concentration}")
```
